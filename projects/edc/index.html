Garden Light System

Automatic intensity adjustment & smart control using LDR Sensors for energy-efficient illumination.

Project Abstract

This project focuses on designing and implementing a garden lighting system with adjustable light intensity, aimed at enhancing both the aesthetics and functionality of outdoor spaces. The system utilizes energy-efficient LED lights coupled with a light-dependent resistor (LDR) to automatically adjust brightness based on ambient light conditions.

Smart Sensing

Brightness adjusts automatically via LDR based on sunlight, ensuring optimal lighting levels without manual intervention.

Energy Efficiency

Significantly reduces power consumption by dimming lights when full brightness isn't required, extending component lifespan.
