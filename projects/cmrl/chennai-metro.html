Simulation Logic

This visualization renders a realistic Metro network state utilizing Vanilla JavaScript and SVG manipulation. It fetches timetable data, calculates train positions via linear interpolation, and updates the view at 60FPS using requestAnimationFrame.

Tech Stack

  • HTML5 SVG + Canvas
  • Vanilla JS (ES6+)
  • Tailwind CSS
