## About the Project

This is a clean, minimal personal homepage designed to provide a professional presence on the web. The focus of this project is on high performance, accessibility, and a seamless user experience across all devices. By hosting on GitHub Pages, the site benefits from automated deployment and high availability.

## Tech Stack

The website is built using core web technologies to ensure lightweight delivery and maximum compatibility:

* **HTML5:** Semantic structure and content.
* **CSS3:** Custom styling, responsive layouts, and modern UI components.
* **JavaScript:** Client-side interactivity and dynamic content rendering.
* **GitHub Pages:** Automated hosting and deployment pipeline.
