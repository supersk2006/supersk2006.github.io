Live Chennai Metro Simulation

A real-time, schedule-based visualization project.

How It Works

This visualization is not a live tracker but a realistic simulation based on the official Chennai Metro timetable. A JavaScript engine running in your browser uses the current time to determine which trains should be running based on the scheduled frequency (peak vs. non-peak hours). It then calculates the precise position of each active train by interpolating its location between stations and renders it smoothly on the SVG map using a `requestAnimationFrame` loop.
